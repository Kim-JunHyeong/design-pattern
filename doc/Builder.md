# Builder Pattern
- 복잡한 단계가 있는 인스턴스 생성 과정 단순화
- **복잡한 단계**를 거쳐야 생성되는 객체의 구현을 **서브 클래스**에게 넘겨주는 패턴
- 복잡한 단계를 추상화시켜서 원하는 알고리즘의 이름을 지어주고 이름을 지어준 알고리즘을 서브 클래스에서 넘겨받아서 설정하는 것


### 학습 목표
- **복잡한 단계**가 필요한 **인스턴스 생성**을 빌더 패턴을 통해서 구현할 수 있다.
