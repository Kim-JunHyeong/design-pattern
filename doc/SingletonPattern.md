# Singleton Pattern
- 하나의 인스턴스만 있도록 하기

### 객체란?
- 객체 : 속성과 기능을 갖춘 것
- 클래스 : 속성과 기능을 정의한 것
- 인스턴스 : 속성과 기능을 가진 것 중 실제하는 것

### 학습 목표
- 싱글톤 패턴을 통해서 **하나의 인스턴스**만 **생성하도록 구현**할 수 있다.
